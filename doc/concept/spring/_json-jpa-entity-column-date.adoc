= Column for type `LocalDate`

== Domain story

This implementation guide applies whenever an entity owns a `LocalDate` property that must be persisted with a specific date format, exposed via the REST and GraphQL APIs, and must not be null.

.Example
....
A visit has a date indicating when the appointment takes place.
The date is stored as a `DATE` column on the `visit` table and is managed as part of the `Visit` entity.
....

<1> The field is marked with `@NotNull` validation constraint.

<2> The field is marked with `@Column`.
The column name in the database is explicitly specified.
The column definition is explicitly specified as `DATE`.
<3> The field is marked with `@Getter`.
This allows the current value to be queried via a getter method.
<4> The field is marked with `@JsonProperty`.
The value is included in the JSON structure of the REST API.
<5> The field is marked with `@JsonFormat` to control JSON serialization format.
The pattern constant defines the date format.
<6> The field has type `LocalDate`.
This is a mutable field due to requirements from the JPA provider.

A `String` constant defines the date pattern for consistent formatting.
A `DateTimeFormatter` is created for parsing and formatting operations; use also in controller or unit tests for consistency.

== Property initialization

The property is typically not nullable and initialized to a default date like `LocalDate.of(2000, 1, 1)` in the constructor.

The `withId` method must copy the property so that cloned instances remain semantically equal.

== Tests

== `<Entity>Test` approach

Create test data from JSON using `fromJson` method of entity classes.
The date value is provided as a string literal in `yyyy-MM-dd` format matching `DATE_PATTERN`.

The `equalsHashcodeToString` test verifies equality operations.

The `withId` test verifies cloning.

The `writeJson` test verifies serialisation to JSON.

The `json<Name>` parameterized test verifies the happy path, e.g.

- `verify` does not throw an exception
- correct deserialization from JSON
- assert property value

.Example
[source,java,options="nowrap"]
----
@ParameterizedTest
@MethodSource
void jsonDate(final LocalDate date) {
    final var value = Visit.fromJson("""
        {
            "date":"%s"
        }
        """.formatted(Visit.DATE_FORMATTER.format(date)));
    assertDoesNotThrow(value::verify);
    assertEquals(date, value.getDate());
}
----
