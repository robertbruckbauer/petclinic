= Invoice REST API

REST-API for managing invoices in the pet clinic application.
It provides standard CRUD operations and partial updates via PATCH, with advanced query capabilities for filtering invoices by text content, invoice date, and due date.

=== `GET /api/invoice`

This operation returns all persisted _Invoice_ entities.

This operation supports advanced query parameters for filtering, e.g.

`?text=Lorem`::
Find invoices with text containing "Lorem" (case-insensitive)
`?text=Lorem%`::
Find invoices with text starting with "Lorem" (case-insensitive)
`?at=2024-01-15`::
Find invoices with invoice date of 2024-01-15
`?at=2024-01-01&at=2024-01-31`::
Find invoices with invoice date between 2024-01-01 and 2024-01-31
`?due=2024-02-15`::
Find invoices with due date of 2024-02-15
`?due=2024-02-01&due=2024-02-29`::
Find invoices with due date between 2024-02-01 and 2024-02-29

This operation supports sorting, e.g.

`?sort=at,asc`::
Sort invoices by invoice date in ascending order
`?sort=due,desc`::
Sort invoices by due date in descending order

This operation supports pagination, e.g.

`?size=10&page=1`::
Find 10 invoices on page 1
